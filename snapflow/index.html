What color should the text in the paragraph be and why?

<style>
	#myID {color:red;}
	.myClass {color:blue;}
	p {color: green;}
</style>
<body>
	<p>id="myID" class="myClass">text inside</p>
</body>

The color will be red because myID has the highest specificity. Class selectors will overrule element selectors and ID selectors will overrules classes.

Why would this code not work as expected? Is anything missing?

function User(name) {
	this.name = name;
};

var j = User('Jack');
alert(j.name);

The function creates an object prototype, and when you create a new User passing in 'Jack' it creates a new object inheriting the properties from the User prototype. In this case we simply need to include the keyword 'new' and it will successfully create a new User object with the name Jack.

Why does the second alert say undefined?

<a id="clickme">click me!</a>
<script>
	var a = document.getElementById('clickme');
	a.onclick = function() {
		alert(this.innerHTML)
		setTimeout( function() {
			alert( this.innerHTML );
		}, 1000);
	};
</script>

Inside the setTimeout function 'this' is referring to setTimeout, which of course has no innerHTML, whereas 'this in the previous function is referring to #clickme.
