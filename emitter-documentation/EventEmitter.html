Class: EventEmitter

EventEmitter

new EventEmitter()

An EventEmitter is responsible for managing a set of listeners and publishing events to them when it is told that such events happened. The following is a very basic EventEmitter class

addListener(event, listener)

Adds a listener to be invoked when events of the specified type are emitted.
Parameters:
Name Type Description
event string Name of the event to listen to
listener function Function to invoke when the specified event is emitted

emit(event, Arbitrary)

Emits an event of the given type with the given data. All handlers of that particular type will be notified.
Parameters:
Name Type Description
event string Name of the event to emit
Arbitrary * arguments to be passed to each registered listener
Source:
  • emitter.js, line 101
Example
emitter.addListener('someEvent', (message) => {
    console.log(message)
  })

  emitter.emit('someEvent', 'hello') // logs 'hello'
